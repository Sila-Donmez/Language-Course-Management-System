# Language Course Management System

A Windows Forms application developed in C# for managing a language course.

## Features
- Role-based login (Student, Teacher, Admin)
- Student grade visualization with charts
- Teacher grade management
- Admin user and class management with statistics
- Password change functionality
- Secure password storage using hashing
- SQLite database integration
- CuoreUI components

## Technologies
- C#
- Windows Forms
- SQLite
- CuoreUI
